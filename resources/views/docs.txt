Postman require : Buka Header lalu masukan (buat nampilin bentuk json)
    key : X-Requested-With
    value : XMLHttpRequest

buat ngepost harus inputin token csrf.
get csrf token : /get-csrf_token
input nya bisa pake key _token . klo dari postman pake header
    key : X-CSRF-TOKEN
    value : (token csrf)

post require : yang harus dimasukan
post optional : data optional, boleh null
