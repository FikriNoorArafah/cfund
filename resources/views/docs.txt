note :

Semua rute di atas kecuali /get-csrf-token memerlukan token csrf untuk request POST. Token csrf bisa didapatkan dengan mengakses rute /get-csrf-token terlebih dahulu. Setelah mendapatkan token csrf, token tersebut harus dimasukkan ke dalam header dengan key 'X-CSRF-TOKEN' dan value berupa token csrf tersebut.
Rute-rute yang hanya dapat diakses setelah login, harus menyertakan token auth pada header. Token auth bisa didapatkan setelah login.
